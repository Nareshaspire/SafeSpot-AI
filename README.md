# Aegis AI — Safety Redefined. AI Driven.

Empowering organizations with next-generation Health & Safety applications and full-lifecycle AI implementation services across all industries.

Built with **React 19**, **TypeScript**, **Tailwind CSS 4**, **Vite**, **Framer Motion**, and **Google Gemini AI**.

## Features

- **AI-Powered Chatbot** — Conversational safety assistant powered by Gemini AI
- **Predictive Risk Assessment** — AI-generated hazard analysis with severity ratings, mitigation strategies, PPE checklists, and regulatory compliance notes
- **Personalized Safety Training** — AI-generated training modules with interactive quizzes and progress tracking
- **Smart PPE Integration** — Biometric monitoring, environmental sensing, gear compliance, and emergency response
- **IEEE Research Standards** — Built on IEEE 7000, 7001, and 2846 frameworks for ethical and transparent AI
- **Dark Mode** — Full dark/light theme with localStorage persistence
- **Lazy-Loaded Sections** — Code-split for fast initial load
- **Responsive Design** — Mobile-first with adaptive navigation

## Getting Started

**Prerequisites:** Node.js >= 20

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file with your Gemini API key:
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```
   Get a free key at [Google AI Studio](https://aistudio.google.com/apikey).

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Scripts

| Command | Description |
